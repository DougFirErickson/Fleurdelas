lasLib__define

lasLib__define

The purpose of this object is to read, manipulate at the point level LAS file. The object is able to handle any LAS 1.0, 1.1, 1.2, 1.3 with point format 0, 1, 2, 3, 4, 5. It also has the ability to write LAS file.

Examples

;Init the object

lasObj = obj_new('laslib')

;Load a LAS file into the Object:

lasObj->loadData, day="206", flightline=5, /QUIET or lasObj->loadDataWithPath, "/path/to/the/las/file", /QUIET

;Getting the bounding box (coverage) of the loaded LAS file:

boxEdge = lasObj->getHeaderProperty(/BOUNDINGBOX)

;Extract data from a geographical area (box):

selectData = lasObj->getData(BOUNDINGBOX=geoBox)

top source lasLib::getData

GENERAL, GET

result = lasLib::getData( [boundingBox=dblarr(6)] [, /max], min=min [, pointNumber=long] [, /all] [, _ref_extra=strarr])

This function returns the number of flightlines of a specific survey day

Return value

An integer equals to the number of flightlines.

Keywords

boundingBox in optional type=dblarr(6)

Geographical limit to filter the data. It can be Easting and/or Northing and/or Elevation values. The array need to be of one of the following format: [xMax, xMin, yMax, yMin, zMax, zMin] or, [zMax, zMin] or, [zMax] or, [zMin]

max in optional type=boolean

Set the boundingBox value as the maximum (cutoff) elevation value. This is a required Keyword if the boundingBox has only one value.

min

Set the boundingBox value as the minimum (cutoff) elevation value. This is a required Keyword if the boundingBox has only one value.

pointNumber in optional type=long

PointNumber can be either one point index (long) and range of continuous points [pointMinIndex, pointMaxIndex], or a collection of discrete points lonarr(n).

all in optional type=boolean

If present, will return all the points of the LAS file.

_ref_extra in optional type=strarr

A strarr[n] that describ the n field(s) that need to be return. If n=0 then all the fields are return.

Examples

Define the bounding coordinate for the selection geoBox = [488401.968647,487901.968647,235421.265389,234921.265386]

Initialize the object lasObj = obj_new('laslib')

Load the 5th flightline from December 5th 2003 data lasObj->loadData, day='338', flightline=5, /QUIET

Select points that lies inside the bounding box. Result = lasObj->getData(boundingBox=geoBox)

Other attributes
